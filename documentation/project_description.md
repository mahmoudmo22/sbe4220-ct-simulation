# Project Title

**Numerical Simulation of 2D Computed Tomography: Acquisition, Reconstruction, and Dose–Quality Trade-offs**

---

## Short Description (April 6th Submission)

This project implements a complete numerical simulation of the 2D X-ray Computed Tomography (CT) pipeline — from physics-based acquisition to image reconstruction. We simulate the forward projection process using the Beer-Lambert attenuation model, generate sinograms under varying acquisition geometries (parallel-beam and fan-beam), and reconstruct images using Filtered Back Projection (FBP). The project explores practical trade-offs between radiation dose and image quality by modeling Poisson photon noise, and investigates the effect of angular undersampling on reconstruction artifacts. We compare the performance of different reconstruction filters (Ram-Lak, Shepp-Logan, Hamming-windowed) and demonstrate a basic iterative reconstruction method (ART) as an alternative to FBP under sparse-view conditions.
